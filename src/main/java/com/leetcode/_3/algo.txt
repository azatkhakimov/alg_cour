1. Создаем мапу символов входной строки и их индексов в строке
2. Создаем переменные int windowStart, windowLength, longest = 0; n - длина входной строки
3. Перебираем в цикле все символы
4. Если текущий символ отсутствует в массиве, добавляем
5. Если присутствует, то значит символ уже видели, тащим из мапы его предыдущий индекс
6. Если индекс предыдущего символа больше или равно windowStart, То обновляем windowLength = i - windowStart;
7. И обновляем longest если оно меньше windowLength
8. И обновляем windowStart = предыдущий индекс повторившегося символа + 1
9. Обновляем в мапе новый индекс текущего символа
10. После обхода, проверяем если longest < n - windowStart, то longest  = n - windowStart
11. Возвращаем результат longest
