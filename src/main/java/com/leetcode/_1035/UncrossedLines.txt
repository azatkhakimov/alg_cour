1. Создаем две переменные, n1 - длина первого входного массива, n2 - длина второго входного массива
2. создаем двумерную матрицу dp длины n1+1, n2+1
3. Итерируемся от i = 1 пока меньше или равно n1
4. Внутренне итерируемся от j = 1 пока меньше или равно n2
5. Если предыдущий элемент из первого массива равен предыдущему элементу из второго массива nums1[i-1] == nums2[j-2],
то dp[i][j] = 1 + dp[i-1][j-1]
6. Иначе dp[i][j] = Math.max[dp[i-1][j],dp[i][j-1];
7. Возваращаем результат dp[n1][n2]

Сложность
n1 длина nums1, n2 длина nums2
Временная сложность:
Заполнение dp заберет O(n1*n2)
